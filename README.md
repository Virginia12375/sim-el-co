# Sim El Co. – Electrical Services Website

Sim El Co. is a modern full-stack website built for a freelance electrician in Nairobi, Kenya.  
The platform showcases electrical services, allows customers to submit reviews, and provides a responsive contact interface.

This project demonstrates full-stack development using **Django, REST APIs, and modern frontend technologies**.

---

## Features

### Responsive Website
- Mobile-first responsive design
- Hamburger navigation menu
- Smooth scrolling interface
- Grid and flexbox layouts

### Customer Review System
- Submit reviews with star ratings (1–5)
- Filter reviews by rating
- Pagination for large review lists
- Server synchronization with **localStorage fallback**

### Django REST API
- RESTful API for managing reviews
- CRUD operations for customer feedback
- SQLite database with Django ORM

### Service Showcase
- Residential electrical services
- Commercial installations
- Industrial electrical solutions
- Animated service cards

### Contact & Requests
- Service request form with validation
- Business contact information
- WhatsApp integration for instant communication
